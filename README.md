📌 Sobre o Projeto

Este projeto consiste na criação de um dashboard interativo desenvolvido no Power BI com o objetivo de analisar o desempenho comercial de uma empresa fictícia.

O painel permite acompanhar métricas essenciais como faturamento, volume de vendas, desempenho por região, vendedores e produtos, facilitando a tomada de decisão baseada em dados.

🎯 Objetivos
Monitorar o faturamento total da empresa
Analisar a quantidade de vendas realizadas
Avaliar o desempenho por vendedor
Identificar regiões com maior e menor faturamento
Acompanhar a média de descontos aplicados
Explorar os dados de forma interativa através de filtros
📈 Principais Indicadores (KPIs)
💰 Faturamento Total
📦 Quantidade de Vendas
📉 Média de Desconto
💳 Tipo de Pagamento
📊 Visualizações do Dashboard
Faturamento por mês (análise temporal)
Faturamento por região
